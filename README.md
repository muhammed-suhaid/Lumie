# 💖 Lumie: For The Ones Who Light Up Your World ✨

### Overview

Lumie is a modern dating app that helps people form meaningful connections through personality-based matching.
The app guides users through an easy onboarding flow — from creating a profile and setting preferences to taking a personality quiz — to ensure better compatibility and authentic matches.


### Key Features

1. **Personality-Based Matching** : Find people who truly match your vibe through a personality quiz.
2. **Multi-Step Onboarding** : Collects user details, relationship preferences, and interests in simple guided steps.
3. **Profile Setup** : Upload photos and videos, edit information, and manage your profile easily.
4. **Discover Screen** : Browse profiles filtered by compatibility and personal preferences.
5. **Chat System** : Seamless one-to-one messaging once a match is made.
6. **Secure Authentication** : Firebase phone authentication with OTP verification.
7. **Media Storage** : Cloudinary integration for smooth image and video handling.


### User Interface Screens

1. **Get Started Screen** : Welcomes users with app branding and introduction.
2. **OTP Verification Screen** : Allows secure login using Firebase Authentication.
3. **Build Profile Screen** : Lets users upload a profile photo and personal details.
4. **Preferences Flow** : Choose goals, who to meet, relationship type, and interests.
5. **Personality Quiz Screen** : Personality-based questionnaire for better matching.
6. **Discover & Chat Screens** : Explore compatible profiles and start chatting.
7. **Settings Screen** : Manage account, privacy, and subscriptions.


### Technology Stack

1. **Flutter & Dart** : Frontend development for smooth and responsive UI.
2. **Firebase Authentication** : Secure OTP-based phone login.
3. **Firebase Firestore** : Real-time database for storing user data and matches.
4. **Cloudinary** : Handles image and video uploads.
